# Netflix Data Analysis Dashboard

This project is an interactive Netflix Data Analysis Dashboard built using Tableau. It enables users to explore Netflixâ€™s movies and TV shows catalog through dynamic filters and visually rich insights.

---
## Project Overview

The dashboard allows users to select between Movies and TV Shows, dynamically updating available titles and displaying detailed metadata for the selected content.

Based on user selections, the dashboard provides insights into:
- Content trends over the years
- Genre popularity
- Ratings distribution
- Country-wise content availability
---

---
## Dashboard Visualizations

- **Total Movies and TV Shows by Year**  
  Shows content growth trends over time.

- **Movies vs TV Shows Distribution**  
  Donut chart comparing overall content types.

- **Top N Genres**  
  Adjustable ranking of the most common genres.

- **Total Movies and TV Shows by Country**  
  World map showing geographic distribution.

- **Ratings Distribution**  
  Count of titles by rating category.
